# 🎓 知识星球 — 高中学习网站 · 完整设计

## 🎯 目标

做一个孩子**主动想打开**的学习网站，不是又一个枯燥的题库。

核心思路：**把学习包装成游戏** — 经验值、等级、成就、打卡、排行榜，每点一下都有反馈。

---

## 🏗 整体架构

```
知识星球
├── 🏠 首页仪表盘    等级/经验/打卡/今日任务
├── 📚 学科中心      6门学科，每科独立进度
├── 🃏 速记卡        知识点卡片，翻转查看
├── 🎯 闯关答题      选择题，连对加分
├── 🏆 成就系统      徽章收集
├── 📊 学习报告      周报/月报
└── ⚙️ 设置         昵称/头像/目标
```

## 🎮 游戏化系统

### 经验值 (XP)
| 行为 | XP |
|------|-----|
| 每日登录 | +5 |
| 翻一张卡片 | +2 |
| 答对一题 | +10~20 |
| 答错一题 | +3（鼓励尝试） |
| 连续 7 天打卡 | +50 奖励 |
| 解锁成就 | +30 |

### 等级
| 等级 | XP 需求 | 称号 |
|------|---------|------|
| Lv.1 | 0 | 🌱 新手学徒 |
| Lv.2 | 100 | 📖 初窥门径 |
| Lv.3 | 250 | ⚡ 小有所成 |
| Lv.5 | 500 | 💎 融会贯通 |
| Lv.8 | 1000 | 👑 知识达人 |
| Lv.10 | 2000 | 🏆 学神降临 |

### 成就徽章（12 个）
| 徽章 | 条件 |
|------|------|
| 🌟 初来乍到 | 首次登录 |
| 🔥 三日之约 | 连续 3 天 |
| 💪 七日之约 | 连续 7 天 |
| 📅 三十之约 | 连续 30 天 |
| 🃏 卡牌新手 | 翻 10 张卡 |
| 🎴 卡牌大师 | 翻 100 张卡 |
| 🎯 初战告捷 | 答对 5 题 |
| 💯 百发百中 | 连续答对 10 题 |
| 🏅 全能选手 | 6 科都学过 |
| 📚 博学多才 | 总分 500+ |
| 👑 学神认证 | 总分 2000+ |
| 🌙 夜猫子 | 22:00 后学习 |

### 每日任务（刷新）
- [ ] 翻 5 张卡片 (+20 XP)
- [ ] 答对 3 道题 (+30 XP)
- [ ] 学习 2 门不同学科 (+15 XP)
- [ ] 完成全部任务 (+50 XP 额外奖励)

## 📱 技术方案

- **纯静态 HTML** — 一个 `index.html`，无框架无构建
- **GitHub Pages** 部署 — 免费，自动 HTTPS
- **localStorage** — 数据存浏览器，不收集隐私
- **响应式** — 手机/平板/电脑都能用
- **PWA** — 可添加到手机桌面，离线也能用
